## Core Concepts

- **IntentKernel**: Capability-secure microkernel replacing permission-based access control
- **Intent Declarations**: Programs declare what they need, kernel mediates access
- **Capability Tokens**: Unforgeable, time-bounded, cryptographically verified access grants
- **UCCS Architecture**: Full-stack OS architecture from kernel to application layer
